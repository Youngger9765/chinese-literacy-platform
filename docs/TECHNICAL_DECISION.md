# 技術決策依據

> 基於「朗朗上口」與「Duotopia」兩大成功平台的建設經驗，快速建置國語文學習平台

---

## 📌 兩大成功平台

本專案的技術決策基於以下兩個已驗證成功的平台：

1. **朗朗上口**（國語文閱讀流暢度分析系統）
   - 與曾世杰教授共同開發
   - 驗證了國語文朗讀分析的教學可行性
   - 提供完整的即時回饋與錯誤訂正機制

2. **Duotopia**（英文學習平台）
   - Young 獨立開發，3 個月完成
   - 已商業運作，獲高中與國中學生接受
   - 驗證了「校、班、生、師、課」架構的技術可行性
   - 已實作朗讀練習、錯誤回饋、即時分析等完整功能

---

## 🚀 開發策略：快速建置上線

### 為什麼不全部重新開發？

1. **⚠️ 資訊不對稱**
   - 兩位高中生對教育領域 Domain Know-how 不完全清楚
   - 需要有經驗的人帶領，避免走冤枉路
   - 朗朗上口與 Duotopia 已踩過的坑，不需要再踩一次

2. **✅ 已有成功範例**
   - 朗朗上口證明：理論可以落地
   - Duotopia 證明：技術可以商用
   - 兩者結合 = 最快的成功路徑

3. **🎯 專注核心價值**
   - 學生應專注在**國語文專屬學習模組**開發
   - 基礎架構、權限、合規等「已驗證可行」的部分直接借鑑
   - 把時間花在刀口上（生字學習、蘇格拉底對話）

### 開發優先順序

#### Phase 1（1-2 個月）：核心架構建立

**目標**：建立穩定的基礎平台

**借鑑 Duotopia 範式**：
- ✅ 校、班、生、師、課多層級資料模型
- ✅ 教師管理介面（班級、學生、課程管理）
- ✅ 學生登入與課程分配
- ✅ 權限管理機制
- ✅ 合規機制（隱私、資料保護）
- ✅ Azure Speech SDK 整合（設定為 zh-TW）

**技術棧**：
- 後端：FastAPI / Node.js + Express
- 前端：Next.js (教師端) + Flutter (學生端)
- 資料庫：PostgreSQL + Redis
- 語音：Azure Speech SDK (zh-TW)
- 部署：Vercel / AWS / GCP

**交付成果**：
- 教師可建立學校、班級、學生
- 學生可登入並查看課程
- Azure Speech SDK 繁體中文語音識別測試通過

---

#### Phase 2（3-4 個月）：國語文學習模組

**目標**：實作國語文專屬功能

**借鑑朗朗上口設計**：
- ✅ 朗讀錄音與流暢度分析
- ✅ 即時回饋系統（六階段）
  - 朗讀結果總覽
  - 錄音播放與轉錄
  - 逐句分析對比
  - 錯字詞練習清單
  - 練習建議
  - AI 詳細分析
- ✅ 錯誤偵測與分類（跳字、加字、讀錯）
- ✅ 統計報告與學習追蹤

**新增功能**：
- 🆕 **生字學習模組**
  - 字型：部件拆解、結構分析
  - 字音：注音符號、發音練習
  - 字義：解釋、例句
  - 筆順：動畫示範
  - 書寫：平板手寫辨識（Google ML Kit）

- 🆕 **蘇格拉底對話系統**
  - LLM 驅動（OpenAI GPT-4o / Claude 3.5 Sonnet）
  - 語音輸入支援
  - 理解力評估
  - 對話歷史紀錄

**交付成果**：
- 學生可完整進行朗讀練習並獲得即時回饋
- 生字學習模組可獨立使用
- 蘇格拉底對話系統可評估理解力

---

#### Phase 3（持續）：優化與擴展

- 📊 進階學習分析
- 🌍 多語言支援（閩南語、客語）
- 👨‍👩‍👧‍👦 家長端（查看孩子進度）
- 📈 AI 個別化學習路徑

## 💰 成本與時程估算

### 開發時程

| 階段 | 時間 | 里程碑 |
|------|------|--------|
| **Phase 1** | 1-2 個月 | 核心架構完成 |
| **Phase 2** | 3-4 個月 | 學習模組完成 |

### 人力成本

```
啟翔：6 個月 × 80 小時/月 = 480 小時
張靖杭：6 個月 × 80 小時/月 = 480 小時
時薪：$500-800/小時

總計：960 小時 × $650 = $624,000
```

### 雲端服務成本（6 個月）

```
AWS / GCP 基礎架構：$100/月
Azure Speech SDK：$200/月（100 位學生使用）
其他服務：$50/月

6 個月總計：$350 × 6 = $2,100
```

### 總成本估算

```
人力：$624,000
雲端：$2,100
其他：$30,000（網域、測試裝置）
-------------------
總計：約 $656,000
```

**vs 全新開發估算**：$900,000-1,200,000
**節省成本**：$244,000-544,000 (27-45%)

## ⚠️ 風險管理

### 風險 1：Azure Speech SDK 中文準確度不足

**機率**：低
**影響**：高

**應對**：
- Phase 1 第一週立即測試繁體中文語音識別
- 準備備選方案（Google Speech-to-Text）
- 與曾教授確認最低可接受準確率標準（建議 > 90%）

### 風險 2：學生對 Domain Know-how 理解不足

**機率**：中
**影響**：中

**應對**：
- Young 每週 Code Review 與指導
- 提供朗朗上口功能說明文件
- 雙週與曾教授對齊確認方向
- 遇到教學理論問題立即詢問曾教授

### 風險 3：開發進度落後

**機率**：中
**影響**：中

**應對**：
- 敏捷開發，每 2 週交付可展示功能
- Phase 1 完成後評估進度，必要時調整 Phase 2 範圍
- 核心功能優先（朗讀分析 > 生字學習 > 蘇格拉底對話）

### 風險 4：使用者接受度低

**機率**：低（有朗朗上口與 Duotopia 驗證）
**影響**：高

**應對**：
- Beta 測試階段密集收集回饋
- 快速迭代 UI/UX
- 提供教師訓練與技術支援
