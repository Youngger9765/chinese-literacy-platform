## 📌 核心理念

**朗朗上口**專注於解決台灣國小學生國語文能力的三大痛點：

1. **朗讀流暢度評估困難** - 教師無法逐一檢查每位學生朗讀
2. **手寫練習缺乏即時回饋** - 學生寫錯了不知道哪裡錯
3. **聽說讀寫功能分散** - 學生需要在多個平台切換

透過 **AI 語音辨識 + 手寫評分 + 個人化學習路徑**，打造台灣首個「聽說讀寫四合一」國語文學習平台。
