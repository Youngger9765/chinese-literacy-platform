#### 3. 曾世杰教授教學理念（核心）

**閱讀方程式**：
```
閱讀能力 = 識字解碼 × 背景知識
```
兩者缺一不可。系統設計必須同時強化「解碼」與「理解」。

**核心教學原則**：

| 原則 | 說明 | 系統如何實踐 |
|------|------|-------------|
| **部件教學法** | 教導孩子拆解國字（如：「清」= 水部 + 青聲） | 生字學習模組顯示部件拆解 |
| **聲韻覺識** | 透過玩聲音強化大腦對文字聲音的敏感度 | 注音符號拼讀練習 |
| **流暢朗讀** | 識字自動化，不需一字一頓 | 追蹤「語速 × 正確率」 |
| **針對性練習** | 避免無效罰抄，只練錯的字 | 錯字詞練習清單（只練錯的字） |
| **早期介入** | 發現落後即刻介入，不等待鑑定報告 | 學習預警通知（超過 7 天沒進展） |
| **馬太效應警示** | 早期識字困難若未解決，三年級後差距擴大 | 系統設計強調一、二年級為黃金期 |

**重要參考連結**：

## 🎯 產品目標

### 問題陳述

**教師面臨的痛點**：
1. ❌ **無法逐一聽每位學生朗讀**
   - 一個班級 25-30 位學生，每位學生朗讀 2 分鐘課文，需要 50-60 分鐘
   - 教師沒有足夠時間逐一聽讀、逐句分析、給予回饋

2. ❌ **難以追蹤每位學生的進步軌跡**
   - 紙本記錄繁瑣且容易遺失
   - 難以產生視覺化的學習曲線圖
   - 無法快速識別需要介入的學生

3. ❌ **缺乏科學化的流暢度評估工具**
   - 僅靠主觀判斷（「感覺進步了」vs「好像退步了」）
   - 沒有客觀的語速、正確率數據
   - 難以設定明確的改進目標

**學生面臨的痛點**：
1. ❌ **不知道自己哪裡讀錯**
   - 朗讀完後沒有即時回饋
   - 需要等待教師批改，錯過最佳學習時機

2. ❌ **缺乏自主練習的動力與方向**
   - 不知道如何改進
   - 沒有看到自己的進步軌跡

3. ❌ **生字學習缺乏系統化支援**
   - 讀錯的字沒有針對性練習
   - 缺乏字型、字音、字義的完整學習

---

### 解決方案

**利用 AI 技術自動化分析學生的國語文朗讀表現，協助教師精準掌握每位學生的閱讀流暢度，實現個別化教學。**

**核心價值主張**：
- 🤖 **AI 代勞**：教師不需逐一聽讀，AI 自動分析語速、正確率、錯誤類型
- 📊 **數據驅動**：客觀的數字取代主觀感覺，學習進步可視化
- ⚡ **即時回饋**：學生練習完立刻獲得診斷報告與改進建議
- 🎯 **個別化教學**：每位學生有專屬目標與學習路徑
- 📈 **可規模化**：一位教師 + AI 系統 = 可同時服務 30 位學生

## 🏢 市場推廣策略與政府合作

### 政府痛點與市場機會

#### 1. 學習扶助計畫（Learning Assistance Program）

**計畫概述**：
- **主管機關**：教育部國民及學前教育署
- **目標對象**：國語文、數學、英語學習低成就學生
- **實施平台**：PRIORI（篩選、追蹤、成長測驗平台）
- **預算規模**：每年數億元，補助全台中小學

**核心痛點**：
1. ❌ **缺乏有效的個別化學習工具**
   - 現行補救教學仍以小班教學為主
   - 教師難以同時照顧不同程度學生
   - 學生學習進度追蹤仰賴人工紀錄

2. ❌ **國語文補救教學資源不足**
   - PRIORI 平台主要提供測驗功能
   - 缺乏即時回饋的朗讀練習工具
   - 缺乏針對性的生字學習模組

3. ❌ **教學成效難以量化**
   - 補救教學成效評估仰賴定期測驗
   - 缺乏學習歷程數據
   - 難以精準定位學生卡點

**我們的解決方案**：
- ✅ **AI 自動化朗讀分析**：減輕教師負擔，提供即時回饋
- ✅ **個別化學習路徑**：系統自動追蹤每位學生的錯誤模式
- ✅ **數據驅動**：完整的學習歷程數據，可視化進步軌跡
- ✅ **針對性練習**：錯字詞自動彙整，提供部件教學

**市場推廣策略**：
```
免費提供工具給學習扶助計畫 → 驗證教學成效 → 申請列入 PRIORI 推薦工具 → 全台推廣
```

**預期成果**：
- 成為「學習扶助計畫」官方推薦工具
- 透過政府管道推廣至全台中小學
- 建立政府背書的公信力

#### 2. 閱讀素養提升計畫

**計畫概述**：
- **政策名稱**：新世代雙閱讀計畫（數位閱讀 + 紙本閱讀）
- **主管機關**：教育部
- **預算規模**：6,900 萬元（2024 年）
- **涵蓋學校**：577 所學校

**核心痛點**：
1. ❌ **閱讀理解評估工具不足**
   - 缺乏即時的流暢度評估
   - 難以追蹤個別學生的閱讀能力發展

2. ❌ **數位閱讀工具品質參差**
   - 市面上工具多為測驗導向
   - 缺乏符合閱讀科學的教學設計

**我們的解決方案**：
- ✅ **符合閱讀科學原理**：曾世杰教授驗證的教學方法
- ✅ **數位 + 紙本結合**：學生朗讀紙本課文，AI 分析數位化
- ✅ **閱讀方程式實踐**：識字解碼（朗讀分析）× 背景知識（蘇格拉底對話）

**市場推廣策略**：
```
提供給閱讀素養計畫學校試用 → 收集教學成效數據 → 申請納入推薦工具清單 → 規模化推廣
```

---

#### 3. 偏鄉教育困境

**現況數據**（教育部統計）：
- **識字率差距**：偏鄉學生中，17% 低於基本識字水準（都市僅 8%）
- **閱讀能力落差**：城鄉差距達 6 年學習進度
- **師資不足**：偏鄉學校難以招募專業國語文教師
- **學習資源匱乏**：缺乏個別化輔導資源

**核心痛點**：
1. ❌ **師資不足，無法逐一聽讀**
   - 偏鄉學校教師流動率高
   - 一位教師常需跨年級教學
   - 無法給予每位學生充分的朗讀練習機會

2. ❌ **學生缺乏課後學習支援**
   - 家庭無法提供學習協助
   - 缺乏課後輔導資源
   - 學習落差逐年擴大（馬太效應）

**我們的解決方案**：
- ✅ **AI 代勞**：彌補師資不足，提供 24/7 學習支援
- ✅ **無需家長介入**：學生可獨立使用，立即獲得回饋
- ✅ **早期介入**：學習預警系統，及時發現需要幫助的學生

**市場推廣策略**：
```
透過 CoLab（誠致 KIST + 均一 + TFT）進入偏鄉學校 → 驗證教學成效 → 成為政府偏鄉教育解決方案 → 擴大至全台偏鄉
```

### 教育組織合作策略

#### 價值交換模型（非資源索取）

**核心原則**：
```
❌ 不向 NPOs 要錢/要資源（他們已經很窮了）
✅ 提供免費工具解決他們的痛點 → 換取推廣管道與社群擴散
```

**價值交換公式**：
```
我們提供：免費使用 + 教學成效提升 + 教師減負
他們提供：推廣管道 + 教師社群 + 使用案例 + 公信力背書
```

**為什麼這個模式有效？**
- NPOs 最缺：**有效的教學工具**（不是錢）
- NPOs 最多：**人力、社群網絡、公信力**
- 我們最缺：**推廣管道、用戶**（初期不缺錢）
- 我們最多：**技術能力、產品**

## 🎓 教學理論驗證（Pedagogy Validation）

### 閱讀科學原則

本產品設計符合以下閱讀科學原則：

1. **流暢度 = 速度 × 準確度**
   - 系統同時追蹤語速與正確率，不偏廢任一指標

2. **即時回饋比延遲回饋有效 10 倍以上**
   - 學生練習完立刻獲得診斷報告

3. **針對性練習比大量練習有效**
   - 系統自動彙整讀錯的字，提供生字學習模組

4. **後設認知能力培養**
   - 學生可重聽錄音、自我覺察錯誤

5. **個別化教學**
   - 每位學生有專屬目標與學習路徑

---

### 曾世杰教授教學方法的系統實踐

#### 1. 閱讀方程式：識字解碼 × 背景知識

**教學原理**：
```
閱讀能力 = 識字解碼 × 背景知識
```
兩者是「乘法」關係，缺一則整體能力為零。

**系統如何實踐**：

| 面向 | 教學目標 | 系統功能 | 驗證方式 |
|------|---------|---------|---------|
| **識字解碼** | 自動化識字、流暢朗讀 | 朗讀練習 + 即時回饋 | 語速 > 150 字/分 + 正確率 > 90% |
| **背景知識** | 理解文意、推理思考 | 蘇格拉底對話系統 | AI 評估理解力得分 |

**設計原則**：
- 不能只練朗讀（解碼）而忽略理解（背景知識）
- 不能只問理解題而不練流暢度（解碼）
- **兩者必須並重**

---

#### 2. 部件教學法

**教學原理**：
教導孩子拆解國字，建立「字感」，讓孩子有「預測」陌生字的能力。

**範例**：
```
「清」= 氵（水部，與水有關）+ 青（聲音）
→ 學生看到「清」可以猜：「跟水有關，讀音可能是『青』」
```

**系統如何實踐**：

**生字學習模組 - 部件拆解功能**：
- **顯示部件**：「清」分解為「氵」+「青」
- **部件說明**：「氵（水部）與水有關，青是聲音」
- **相關字延伸**：「清、情、晴、請」都有「青」這個聲符
- **互動練習**：「請找出下列哪些字與『水』有關？」

**Acceptance Criteria**：
- [ ] 每個生字都必須顯示部件拆解
- [ ] 部件說明必須包含「形符」（意義）與「聲符」（讀音）
- [ ] 提供至少 3 個相關字延伸學習

---

#### 3. 聲韻覺識（Phonological Awareness）

**教學原理**：
透過「玩聲音」（拼音、拆音、換音）強化大腦對文字聲音的敏感度。

**範例**：
```
拼音：ㄑ + ㄧ + ㄥ = 清
拆音：「清」可以拆成 ㄑ、ㄧ、ㄥ
換音：「清」的 ㄑ 換成 ㄊ，變成「聽」
```

**系統如何實踐**：

**注音符號拼讀練習**（P1 功能）：
- **顯示注音**：「清」→ 「ㄑㄧㄥ」
- **拼讀遊戲**：「請拼出『清』的注音符號」
- **換音練習**：「把『清（ㄑㄧㄥ）』的 ㄑ 換成 ㄊ，會變成什麼字？」
- **隨時開關**：學生可用 Space 鍵切換注音符號顯示

**Acceptance Criteria**：
- [ ] 注音符號可隨時開關（Space 快捷鍵）
- [ ] 懸浮顯示於文字上方（不影響閱讀）
- [ ] 生字學習模組包含拼讀練習

---

#### 4. 流暢朗讀的重要性

**教學原理**：
識字自動化（不需一字一頓），大腦才有餘裕處理文意。

**馬太效應警示**：
- **優勢學生**：識字快 → 流暢朗讀 → 大量閱讀 → 背景知識增加 → 越讀越好
- **弱勢學生**：識字慢 → 一字一頓 → 無法享受閱讀 → 背景知識貧乏 → 越來越差

**系統如何實踐**：

**追蹤「語速 × 正確率」**：
- **語速目標**：150-180 字/分（識字自動化的標準）
- **正確率目標**：> 90%（準確度不能犧牲）
- **防止「快而不精」**：若語速 300 字/分但正確率 50% → 未達標
- **防止「慢而準」**：若正確率 100% 但語速 80 字/分 → 識字未自動化

**Acceptance Criteria**：
- [ ] 系統同時追蹤語速與正確率
- [ ] 兩者都必須達標才顯示✅
- [ ] 達標標準可由教師個別調整（因材施教）

---

#### 5. 針對性練習（避免無效罰抄）

**教學原理**：
對於讀障者，機械式的重複抄寫只會帶來挫折，應改用**多感官與部件教學**。

**反例（無效罰抄）**：
```
❌ 學生讀錯「清」→ 教師罰抄「清」100 次
→ 學生只記得「痛苦」，沒記得字怎麼唸
```

**正確做法（針對性練習）**：
```
✅ 學生讀錯「清」→ 系統自動加入錯字詞清單
→ 提供🔊發音按鈕（聽正確唸法）
→ 顯示部件拆解（氵 + 青）
→ 相關字練習（清、情、晴、請）
```

**系統如何實踐**：

**錯字詞練習清單**：
- **自動彙整**：系統自動抓取所有讀錯的字
- **發音按鈕**：點擊🔊即可聽正確唸法
- **部件教學**：顯示部件拆解與說明
- **多感官學習**：聽（發音）、看（部件）、寫（筆順）

**Acceptance Criteria**：
- [ ] 所有讀錯的字自動加入清單
- [ ] 每個錯字提供發音按鈕
- [ ] 每個錯字顯示部件拆解
- [ ] 可點擊進入完整生字學習模組

---

#### 6. 早期介入（馬太效應的預防）

**教學原理**：
發現落後即刻介入，不需要等待醫學鑑定報告。**一、二年級是識字發展的黃金期。**

**馬太效應的臨界點**：
- **三年級前**：還有機會追上
- **三年級後**：差距會因「無法透過閱讀獲取知識」而快速擴大

**系統如何實踐**：

**學習預警通知**：
- **超過 7 天沒練習** → 自動通知教師
- **連續 3 次未達標** → 自動通知教師
- **同一生字錯誤 ≥ 3 次** → 卡點提醒
- **提醒內容具體**：「小明在『清』這個字錯了 3 次，建議進行部件教學」

**Acceptance Criteria**：
- [ ] 系統每日自動檢查學生學習狀態
- [ ] 符合條件立即通知教師
- [ ] 通知內容包含具體行動建議
- [ ] 教師可設定通知頻率（避免過度打擾）

---

#### 7. 後設認知能力培養

**教學原理**：
優秀學習者的共同特質：會自我監控、自我調整。**「我知道我哪裡不會」是學習的關鍵。**

**系統如何實踐**：

| 功能 | 目的 | 培養的能力 |
|------|------|-----------|
| **重聽錄音** | 學生聽到自己的聲音 | 覺察「我剛剛讀錯了」 |
| **逐句對比** | 原文 vs 實讀 | 精準定位錯誤 |
| **學習曲線圖** | 10 次練習的折線圖 | 看到進步軌跡 |
| **錯字詞清單** | 彙整所有錯誤 | 建立「錯誤清單」習慣 |

**Acceptance Criteria**：
- [ ] 每次練習都可重聽錄音
- [ ] 逐句對比清楚標示錯誤
- [ ] 學習曲線圖顯示長期趨勢
- [ ] 錯字詞清單可下載（供家長參考）
