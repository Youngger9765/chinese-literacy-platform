# 曾世杰教授閱讀教學實作手冊

> 整理自曾世杰教授的教學理論與實務經驗

## 第一部分：核心教學理念

### 基本信念

**核心精神**：閱讀不是天賦，而是可以透過「有效教學」習得的技能。我們必須透過科學證據（Evidence-based）來縮小學習落差。

### 馬太效應的警示

早期識字困難若未解決，孩子在三年級後會因無法透過閱讀獲取知識，導致貧富差距式的學習落差。

### 閱讀方程式

```
閱讀能力 = 識字解碼 × 背景知識
```

兩者缺一不可。

### 平衡取向

教學應兼顧：
- ✅ **自動化的識字訓練**
- ✅ **高層次的意義理解**

## 第二部分：如何判斷孩子是否有「讀寫障礙」？

### 觀察指標檢查表

#### 1. 識字自動化困難
- [ ] 學過的字容易忘
- [ ] 閱讀流暢度極低
- [ ] 讀起來一字一頓

#### 2. 視覺與語音轉換問題
- [ ] 容易讀錯形近字（如：由/甲）
- [ ] 容易讀錯音近字

#### 3. 拼讀基礎薄弱
- [ ] 到了中高年級，注音符號拼讀仍不流暢

#### 4. 閱讀行為異常
- [ ] 容易跳行、漏字
- [ ] 用「猜讀」的方式讀書

#### 5. 能力落差明顯
- [ ] 口語表達、理解力正常
- [ ] 唯獨「讀」與「寫」有嚴重困難

### 曾老師的具體建議

#### ⚠️ 不要等待
發現落後即刻介入，不需要等待醫學鑑定報告。

#### ❌ 避開無效罰抄
對於讀障者，機械式的重複抄寫只會帶來挫折，應改用多感官與部件教學。

## 第三部分：具體實作做法

### 1. 識字與解碼策略

#### 部件教學法
教導孩子拆解國字：
```
例如：「清」= 水部（與水有關）+ 青（聲音）
```
建立字感，讓孩子有「預測」陌生字的能力。

#### 聲韻覺識遊戲
透過玩聲音（拼音、拆音、換音）來強化大腦對文字聲音的敏感度。

#### 字母拼讀（英文）
採用「直接教學法」（Direct Instruction）：
- ⏱️ 每日 15 分鐘小步快跑
- 🎯 強調精熟練習

### 2. 背景知識與理解

#### 運用鷹架（如：漫畫）
曾老師研發《漫畫語文故事集》：
- 利用圖像降低認知負荷
- 讓識字量少的孩子也能理解複雜邏輯
- 維持學習動機

#### 大量朗讀（Read Aloud）
家長唸書給孩子聽：
- 幫助孩子接觸比他認字程度更高的內容
- 接觸更豐富的書面語彙

---

## 第四部分：進階引導——邁向長篇小說

當孩子從繪本轉入文字小說時，請給予以下支持：

### 五指原則選書
隨機翻開一頁，超過 5 個生字則太難，應選擇「合宜難度」的書籍。

### 建立先行組織者
讀長篇前，先：
- 📹 看過相關影片
- 📚 讀過相關漫畫
- 🎧 聽過導讀

讓大腦先有「地圖」。

### 共讀三部曲

#### 1. 預測
從封面猜劇情

#### 2. 澄清
遇到難詞大人直接解釋，不中斷閱讀快感

#### 3. 摘要
讀完一個段落，請孩子用自己的話回饋

---

## 第五部分：給家長與老師的行動清單

- [ ] **建立環境**：家中隨處有書，家長也要有閱讀的背影
- [ ] **早期介入**：一、二年級是識字發展的黃金期，請密切關注
- [ ] **接納多元**：漫畫、小說、雜誌都是好的閱讀起點，興趣優先於深度
- [ ] **正向鼓勵**：對於讀寫困難的孩子，肯定他們的努力而非成績，保護其自尊
