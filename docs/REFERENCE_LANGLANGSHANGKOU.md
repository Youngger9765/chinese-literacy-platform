## 📊 功能流程

### 學生使用流程（推測）
```
1. 登入系統
2. 選擇課文
3. 開始朗讀（錄音）
4. AI 分析朗讀表現
5. 顯示流暢度報告
6. 提供改善建議
```

```

2. 查看學生朗讀紀錄
3. 檢視流暢度分析報告
4. 追蹤學習進度
```
